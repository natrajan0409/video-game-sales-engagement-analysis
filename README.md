# Video Game Sales and Engagement Analysis

> End-to-end data analytics project that merges game engagement data (ratings, plays, wishlists) with global sales records to uncover trends in genre popularity, platform performance, publisher dominance, and regional market behavior.

**Domain:** Gaming and Entertainment Analytics  
**Skills:** Python · SQL · Data Cleaning · EDA · Streamlit · Data Visualization  
**Institution:** GUVI / HCL Capstone Project

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/video-game-sales-engagement-analysis.git
cd video-game-sales-engagement-analysis
```

### 2. Create a virtual environment
```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Set up the database & clean data
```bash
python setup_db.py
```
This will:
- Clean both raw datasets
- Save cleaned CSVs to `Cleandata/`
- Populate `videogames.db` (SQLite)

### 5. Launch the Streamlit dashboard
```bash
# Windows shortcut
run_app.bat

# OR manually
streamlit run streamlit_app/app.py
```

### 6. Run the EDA notebook
```bash
jupyter notebook notebooks/02_complete_eda.ipynb
```

## Key Insights

- **North America** accounts for the largest share of global game sales (~49%)
- **PS2** is the best-selling platform of all time in this dataset
- **Action** and **Sports** genres dominate sales; **RPG** leads in Japan
- User ratings show only a **weak positive correlation** with sales
- **Wishlist count does not reliably predict sales** — engagement ≠ purchases
- Peak game releases occurred around **2008–2009**; sales peaked around **2008**
- Genres like **Adventure** and **Indie** show high engagement but relatively low sales — potential untapped market
